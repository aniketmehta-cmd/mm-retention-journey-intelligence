What this tool does

A reusable audit workspace for CleverTap retention journeys. Drop in your journey CSV exports — for any product, any category, any time period — and the dashboard rebuilds itself from the data. It only shows what's actually in your files: no auto-generated recommendations, no fabricated insights.

01

What you can put in

Any CleverTap journey export CSV — one file or many. The tool dedupes across files automatically (handles the iOS/Android row-split that CleverTap exports for some journeys). Drop them anywhere on the page, or click "Load CSV files" above.

02

What you get out

Per-journey deep-dives (every message, full copy, all the metrics), topline aggregates, a searchable inventory, and auto-generated Data Quality flags pointing at things like duplicate journey names, running-but-silent journeys, and push-fatigue risk.

03

What it doesn't do

It doesn't write recommendations for you. Auto-generated insights are limited to mechanical rules over your data (e.g. "these journeys show as Running but have zero sends"). Bespoke strategic recommendations are a separate exercise per product.

04

How sharing works

This is a static page. Email the bundled HTML to anyone — they can open it in any browser and load their own CSVs. Loaded data stays in their browser only (via local storage); nothing is uploaded anywhere.

How to use it — three steps

  1. Open this file in any browser. Right now you're looking at the Hair / Stage 2 sample data so the tool isn't empty. The orange "DEMO" badge in the bar above marks this. No installation required. Works in Chrome, Safari, Firefox, Edge.
  2. Drop your CleverTap CSV files anywhere on the page — or click "+ Load CSV files" in the bar above. Drop multiple files at once; the tool will dedupe across them. The DEMO badge flips to UPLOADED, and the dashboard rebuilds. Expected columns: Journey ID, Journey Name, Version Status, Node ID, Node Type, Total Sent, Total Delivered, Total Viewed, Total Clicked, Unique Converted within Conversion Time, Influenced Conversions, Influenced Revenue, Goal 1/2/3 Name + Conversions, Campaign Message, Campaign / Segment / Controller Name + Type, Campaign OS, Delay before Node, Connector from Previous. (Standard CleverTap export schema — should match out of the box.)
  3. Browse the tabs. Overview for the topline, Journeys for the full searchable list with deep-dives, Data Quality for auto-flags, Methodology for what each metric means. Your loaded data persists in your browser's local storage, so you don't have to re-upload every time. Click "Reset to demo" to clear and go back to the sample data.

What this tool does NOT replace

For bespoke strategic recommendations (a consolidated journey design for a specific product, or a copy-rotation proposal), this tool surfaces the inputs but the synthesis is a human exercise. Frequency-per-customer caps need to be cross-checked in CleverTap's Global Campaign Limits settings — not exported here. See the Methodology tab for notes on control group data and metric definitions.

How to use this dashboard

A quick-start reference for anyone opening this tool for the first time, plus a full explanation of how the data is processed.

Reading the data

01

What is a Journey?

A CleverTap Journey is an automated multi-channel communication sequence triggered when a customer enters a defined segment or takes an action. Each journey contains multiple messages (nodes) across channels like WhatsApp, Push, and SMS — sent in sequence with configurable delays between each step.

02

Reading Journey Cards

Click any journey to expand it. You'll see: (1) a channel summary bar, (2) a visual flow of every message in order with timing labels, (3) metrics per message node including delivery, read, and conversion rates.

03

Metric Definitions

Sent: attempts to send. Delivered: confirmed delivery (WA/SMS only — Push not tracked this way). Viewed: push impressions + WA read receipts. Clicked: push taps. Converted: purchased within the conversion window. Influenced Conv/Rev: broader attribution including assists.

04

Status Meanings

Running: actively enrolling new users. Paused: no new enrollments, existing users continue. Restricted: entry closed, no new users. Draft: never published.

05

Channel Colors

WhatsApp green  ·  Push blue  ·  SMS amber  ·  Webhook grey. Consistent across flow cards, pills, and channel breakdown tables.

06

Why Totals Look Large

All numbers are lifetime totals since the journey was published. A 2-year-old journey will always outweigh a 2-month-old one on absolute volume. Always check the start date before comparing journeys.

07

Control Group Note

Control group data is not in CleverTap's journey CSV export — it lives in the Campaign Report. For target vs control conversion rates, export the Campaign Report and match by Campaign ID.

08

Adding Data

Click + Add data in the top bar. Select the category and sub-category, then drop in your CleverTap journey CSV exports. Data is stored in your browser — to share with the team, commit the updated file to GitHub.

How the data is processed

All processing happens in your browser. Nothing is sent to a server.

  • Dedup within file: CleverTap splits the same node into Android + iOS rows. The tool groups by (Journey ID, Node ID) and sums metrics across both — giving one correct row per node.
  • Dedup across files: If the same journey appears in multiple uploaded files, the first file loaded wins. Others are dropped silently.
  • Sent — message attempts for comm nodes; trigger fires for webhook nodes (not a customer message).
  • Delivered — WhatsApp and SMS only. Push delivery is not exported by CleverTap in the journey schema.
  • Converted — unique customers who fired the conversion event within the configured window. Direct attribution only.
  • Influenced Conversions / Revenue — broader attribution including post-window and assist conversions. This is typically the headline number.
  • Control-Group lift — not in this export. Get from CleverTap's Campaign Report.
  • Frequency per customer — not computable here. These are journey-level aggregates.
  • Dates — CleverTap exports don't include the year. Dates are shown as-is from the export.
